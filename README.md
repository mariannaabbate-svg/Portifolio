# Documentação do Projeto: Portfólio Profissional

## Introdução

Este documento descreve o desenvolvimento do portfólio pessoal de Marianna Schmidt, um projeto de lad page concebido dentro da trilha de formação SCTEC em Desenvolvimento de Sistemas. O projeto visa consolidar competências em desenvolvimento front-end, integrando princípios de design responsivo, interatividade programática e organização de ativos digitais.

A proposta central deste trabalho é demonstrar os conhecimentos adquiridos para  fundamentar a construção de soluções de software estruturadas e eficientes.

## Especificações Técnicas

O desenvolvimento foi pautado na utilização de tecnologias web padronizadas, garantindo compatibilidade e performance. A tabela abaixo detalha a stack tecnológica empregada:

| Categoria | Tecnologia | Aplicação no Projeto |
| :--- | :--- | :--- |
| Estruturação | HTML | Organização semântica do conteúdo e acessibilidade. |
| Estilização | CSS | Implementação de layouts via Flexbox e Grid, além de Media Queries. |
| Lógica | JavaScript | Controle de estado do Modo Escuro e manipulação do DOM. |
| Versionamento | Git | Controle de versões e publicação via GitHub. |

## Funcionalidades Implementadas

### Gerenciamento de Tema (Modo Escuro)
O sistema possui uma funcionalidade de alternância de tema (Dark Mode) que utiliza a API de LocalStorage do navegador. Isso permite que a preferência do usuário seja armazenada e recuperada automaticamente em sessões futuras, proporcionando uma experiência personalizada e confortável.

### Responsividade e Adaptabilidade
A interface foi projetada sob a filosofia de design responsivo, assegurando que o conteúdo seja renderizado de forma otimizada em dispositivos móveis, tablets e desktops. A navegação é adaptada dinamicamente, utilizando um menu  para telas menores.

### Elementos de Interatividade
*   **Animação de Digitação:** Implementada via JavaScript para destacar o título principal da página.
*   **Navegação Suave:** Utilização de comportamento de rolagem suave (smooth scroll) para âncoras internas.
*   **Efeitos de Transição:** Aplicação de animações de entrada (fade-in) baseadas no scroll do usuário para aumentar o engajamento visual.
